We should be prepared for an emergency situations and model evacuation process. Rooms in the building are connected by corridors various length. And each room has various quantity of staffs in it. We should calculate how many people will be in evacuation room for the given time.

A building room and passage information can be represented as a matrix (M). Each room has a number from 0 to N. The matrix has size NxN. In diagonals we are given information about how many people in the room -- M[i][i] is how many people in the i-th room.
In other cell of matrix are given an information about passages -- how much time (in minutes) are required to pass from room i to room j (M[i][j] == M[j][i]). If time is zero, then passage is not exist between these rooms. The evacuation room is a room with number 0 (zero).

You are given a building matrix and evacuation time (minutes). People use the shortest path for evacuation. How many people will be escaped? The room pass don't take time, only passages. The room will be escaped if total time to evacuation room is less or equal than the given time.

For example look at the image 

[[ 0, 40,  0, 40,  0,   0],
 [40,  6,  0,  0, 40,   0],
 [ 0,  0,  3,  0, 28,   0]
 [40,  0,  0,  4, 40,  28]
 [ 0, 40, 28, 40,  1,   0]
 [ 0,  0,  0, 28,  0,   2]]

It the maximum time is 80, then rooms 1, 3, 4 and 5 will be escaped and the result is 6+4+1+2=13.

Input: Two arguments. Information about a building as a list of lists of integers and evacuation time as an integer.

Output: How many people will escape as an integer.

How it is used: This model can be used for various cases. For example for flood or gathering resources and etc.
